Magnum Instruction Manual

Magnum is operated from the command line on both Windows and Linux systems. Prior to Magnum analysis, data files must be converted to standard open formats, and a sequence database must be generated for the proteins to be searched.

Specific details for setting up, running, and interpreting results from Magnum are in found in the menu on the right.
