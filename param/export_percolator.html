Parameter: export_percolator

Overview

Indicates creating additional export files for use as input to Percolator, a 3rd party PSM validation tool. Additional export file names are derived from the input file name.

Syntax

[Number]

Versions

1.0.0 - present

Example

export_percolator = 1
